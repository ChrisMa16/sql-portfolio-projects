# Layoffs Exploratory Data Analysis

This project uses SQL to analyze a layoffs dataset.

## Skills Used
- SELECT statements
- WHERE filtering
- GROUP BY
- ORDER BY
- Aggregate functions
- Date functions
- CTEs
- Window functions
- DENSE_RANK()
- Rolling totals

## Project Goals
- Find companies with the highest layoffs
- Analyze layoffs by industry, country, and year
- Track monthly layoffs
- Create rolling totals
- Rank companies by layoffs per year
