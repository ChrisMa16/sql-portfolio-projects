## Projects

This project uses SQL to explore and analyze a layoffs dataset. The analysis looks at layoff trends by company, industry, country, year, and month.

Skills demonstrated:
- SELECT statements
- WHERE filtering
- GROUP BY
- ORDER BY
- Aggregate functions
- Date functions
- CTEs
- Window functions
- DENSE_RANK()
- Rolling totals
